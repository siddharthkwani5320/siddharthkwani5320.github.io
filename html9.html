HTML Image Tag

The img tag is used to embed an image in an HTML page. Images are not technically inserted into a web page; images are linked to web pages. The tag creates a holding space for the referenced image. The img tag has two required attributes:

  • src - Specifies the path to the image
  • alt - Specifies an alternate text for the image, if the image for some reason cannot be displayed
Example 1:Image 1
<img scr="image1.jpeg" alt="Exo Band Image">

Example 2:Image 2
<img scr="image2.jpeg" alt="Exo Band Image">

Example 3:Image 3 Does Not Exist
<img scr="image1.jpeg" alt="Exo Band Image">
